
# Kunkunshi Editor

*kunkunshi-editor-okinawa-sanshin-partitions*

The aim of this project is to provide an easily available editor for 沖縄 Okinawa, 琉球 Ryukyu, 三線 Sanshin partitions based on the traditional 工工四 Kunkunshi notation.
