
# Kunkunshi Editor

*kunkunshi-editor-okinawa-sanshin-partitions*

The aim of this project is to provide an easily available editor for Okinawa, Ryukyu, Sanshin partitions based on the traditional Kunkunshi notation.

## References:
1. https://en.wikipedia.org/wiki/Kunkunshi
1. https://en.wikipedia.org/wiki/Sanshin
